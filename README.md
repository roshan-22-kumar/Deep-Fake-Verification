## Overview
This project detects and verifies deepfake media using AI. It analyzes facial features and deep learning outputs to identify manipulated content.

## Features
- Upload and verify videos/images  
- Real-time face detection  
- AI-based authenticity score  
- Simple Flask web interface  

## Tech Stack
- Python (Flask)  
- OpenCV  
- Deep Learning (CNN)  
- HTML, CSS, JavaScript  
- SQLite Database  

## How to Run
1. Install Python and required libraries from `requirements.txt`  
2. Run the app:python app.py
3. Open `http://127.0.0.1:5000/` in your browser
